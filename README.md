# Kinka

Boutique e-commerce de mangas, conçue de A à Z dans le cadre de ma formation DWWM.
Maquette Figma, intégration HTML/CSS responsive, interactions JS — sans framework.

## Stack

HTML5 · CSS3 · JavaScript ES6+ · Figma · Git

## Fonctionnalités

**Catalogue**
- Filtres dynamiques multi-critères (genre, auteur, prix) sans rechargement de page
- Moteur de recherche instantané sur le titre et l'auteur
- Pages dédiées : nouveautés, promotions, coups de cœur, coffrets, catégories

**Panier**
- Ajout/suppression, calcul du total en temps réel, gestion des quantités
- Tunnel complet : panier → paiement → confirmation → suivi de commande

**Fiches produits**
- Détail série et maison d'édition
- Navigation orientée conversion

**Compte utilisateur**
- Inscription, connexion, réinitialisation de mot de passe, profil

**Légal & support**
- CGU, CGV, politique de retour, FAQ, contact
